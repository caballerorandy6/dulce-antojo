# Dulce Antojo

Landing page for **Dulce Antojo** - Mexican dessert and snack cart service for events in Houston, TX.

## Tech Stack

- **Next.js 15** - React framework with App Router
- **React 19** - UI library
- **TypeScript** - Type safety
- **TailwindCSS 4** - Styling
- **shadcn/ui** - UI components
- **React Hook Form + Zod** - Form handling & validation
- **Resend** - Email service

## Features

- Conversion-optimized landing page with video hero
- Contact form with email integration
- Local SEO optimized for Houston area
- Fully responsive design
- Subtle animations and floating stickers
- Server Components for optimal performance

## Services

Mini Pancakes, Churros, Paletas, Churro Sundaes, Elote Cups, Tosti-Elote, Snack Cups, Fresa Cups, Ramen, Sorbet, Sundaes, and Mix & Match packages.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
